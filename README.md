## ✨ Features

* **Elder Futhark Runes:** The clock face uses Norse runes that smoothly transition into English numbers when hovered.
* **Weapon-Inspired Hands:** The hour hand mirrors the Leviathan Axe, while the second hand glows with the fiery red of the Blades of Chaos.
* **Frozen Blue Soul Cursor:** A custom-built, physics-based mouse cursor that trails a glowing blue orb, mimicking the spirits of Helheim.
* **High-Performance Animation:** Utilizes `requestAnimationFrame` for buttery-smooth, continuous hand rotation—no CSS transition jumping.
* **Ambient Particle System:** A lightweight HTML5 Canvas particle system generates background embers and frost.
* **Antique Light Mode:** Toggle between the default dark cavern aesthetic and a bright, aged parchment artifact mode.

## 🚀 Getting Started

No build tools or heavy frameworks are required. This project runs directly in the browser.

1. Clone the repository:
   ```bash
   git clone [https://github.com/ashishbtech/norse-artifact-clock.git](https://github.com/ashishbtech/norse-artifact-clock.git)
   ```
2. Navigate to the project folder:
   ```bash
   cd norse-artifact-clock
   ```
3. Open `index.html` in any modern web browser.

## 📂 File Structure

* `index.html` - The core layout and canvas structure.
* `styles.css` - Custom properties, theme toggling, and layout formatting.
* `script.js` - Real-time clock logic, DOM manipulation, cursor physics, and particle generation.
